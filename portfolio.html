🎯 Proyek Utama: LabPinjam

LabPinjam — Sistem Manajemen Peminjaman Alat Laboratorium

Platform digital terintegrasi untuk mengelola proses peminjaman dan pengembalian alat laboratorium di universitas. Sistem ini mendukung 3 role pengguna (Admin, Petugas, Peminjam) dengan workflow lengkap, dashboard analytics, notifikasi real-time, dan laporan komprehensif.

Tech Stack:

Laravel 11 React 19 MySQL Tailwind CSS Vite GSAP

Fitur Utama:

  • ✓ Sistem autentikasi dengan role-based access control
  • ✓ CRUD lengkap untuk alat, kategori, dan user management
  • ✓ Workflow peminjaman: Pengajuan → Persetujuan → Pinjam → Pengembalian
  • ✓ Sistem denda otomatis untuk keterlambatan dan kerusakan
  • ✓ Dashboard analytics dengan chart interaktif untuk setiap role
  • ✓ Notifikasi real-time untuk peminjaman, pengembalian, dan denda
  • ✓ Import/Export data Excel untuk bulk management
  • ✓ Laporan PDF untuk bukti peminjaman dan statistik
  • ✓ Landing page modern dengan animasi smooth scroll
  • ✓ Mobile-responsive design untuk semua perangkat

Database Schema:

  • ✓ 9 tabel utama (users, tools, categories, borrowings, returns, dll)
  • ✓ Relasi one-to-many dan many-to-many yang terstruktur
  • ✓ Migrations dan seeders untuk setup otomatis
  • ✓ Activity logging untuk audit trail

User Roles & Permissions:

  • ✓ Admin: Full access - kelola semua aspek sistem
  • ✓ Petugas: Operasional - setujui peminjaman, proses pengembalian
  • ✓ Peminjam: End user - lihat alat, ajukan peminjaman, riwayat
